📖 Feature Guide

Everything Easy Redirect can do — tap any feature to see how it works and where to find it.

Blocking websites CORE
📋 Rules tab → Block a Website

Type a website into the Block a Website input and click + Block. The extension will redirect anyone visiting that site to your chosen redirect URL (default: Google).

  • facebook.com — blocks facebook.com and all subdomains (m.facebook.com, etc.)
  • reddit.com/r/funny — blocks only that subreddit, not all of Reddit
  • *.ads.com/* — wildcard pattern, blocks any URL matching the pattern
  • kw:gambling — blocks any URL whose address contains the word "gambling"
  • r/casino.*\.com — regex pattern (max 10 active at once)
💡 Tap ▸ Advanced formats under the input for a full syntax reminder.
Redirect URL — where blocked sites go CORE
⚙️ Settings tab → Redirect URL

The global redirect URL is where every blocked site sends the visitor. Default is https://www.google.com.

You can override this at three levels (most specific wins):

  • Global — Settings → Redirect URL
  • Per group — click a group tab, then set its redirect URL in the field that appears below the tabs
  • Per rule — click ⋯ on any rule row and fill in "Redirect to:"
Enable / Disable the extension CORE
📋 Sidebar — green/red button at the top

The big button at the top of the sidebar toggles blocking on and off. When disabled, all redirect rules are removed from Chrome — sites load normally. Your rule list is kept; re-enabling restores everything instantly.

Keyboard shortcut: Ctrl+Shift+Y (customisable in Advanced → Keyboard Shortcuts).

Groups — organise rules into categories
📋 Rules tab → group tabs row

Rules live inside groups. The Default group always exists. Click + New Group to create more (e.g. "Work", "Social Media").

Hover over any group tab to reveal its controls:

  • ⏰ — set a time schedule (active only on certain days/hours)
  • ⏱ — set a countdown delay before the redirect fires
  • ⏸ — disable the whole group without deleting it
  • × — delete the group (its rules move to Default)
💡 Each group can have its own redirect URL, schedule, and delay.
Prebuilt categories — add 40+ sites in one click
📋 Rules tab → Prebuilt Categories (scroll down)

Six ready-made lists: Social Media, News & Media, Video Streaming, Gaming, Adult, Gambling. Click Add all on any card to create a new group containing all the domains in that list. Duplicates are skipped if you already have them.

Bulk add — paste many sites at once
📋 Rules tab → Bulk Add Rules (scroll down)

Paste one website per line into the textarea, choose a group from the dropdown, and click Add All. Duplicate entries are skipped automatically.

Per-rule exceptions — allow a sub-page inside a blocked site
📋 Rules tab → rule row → ⋯ → Exceptions

Say you block reddit.com but still need reddit.com/r/learnprogramming. Click ⋯ on the reddit.com rule, then + Add under Exceptions and enter the URL pattern to whitelist. That pattern will always pass through even while the parent rule is active.

Enable / disable individual rules
📋 Rules tab → rule row → On / Off button

Every rule has an On / Off button. Turning it off keeps the rule saved but pauses it — Chrome stops redirecting that site until you turn it back on. No re-typing needed.

Daily limit — auto-pause a rule after N redirects per day
📋 Rules tab → rule row → ⋯ → Daily limit

Enter a number in Daily limit. Once the rule has fired that many times today, it automatically suspends itself until midnight (UTC). The limit resets every day. Leave blank for no limit.

💡 Useful for "allow me to visit Twitter up to 5 times a day, then block it".
Temporary allow — pause a rule for 5 / 15 / 60 minutes
📋 Rules tab → rule row → ⋯ → Temp allow

Click ⏸ Allow and choose 5 minutes, 15 minutes, or 1 hour. The rule goes silent for that window — the site loads normally. After the time expires, blocking resumes automatically. An orange badge on the rule row shows how many minutes remain.

Move a rule to a different group
📋 Rules tab → rule row → ⋯ → Move to

Click ⋯ on a rule and use the Move to dropdown to change which group it belongs to. The rule instantly appears under the chosen group tab.

Group schedule — block only on certain days/hours
📋 Rules tab → hover a group tab → ⏰

Click the ⏰ icon that appears when you hover a group tab. Choose which days of the week the group is active, plus a start and end time window. Outside that window the rules are automatically paused — no manual toggling needed.

Example: "Block social media Mon–Fri 09:00–17:00 only."

💡 End time earlier than start time wraps midnight (e.g. 22:00–06:00 = nights only).
Redirect delay / countdown — show a timer before blocking
📋 Rules tab → hover a group tab → ⏱

Set a countdown in seconds. Instead of an instant redirect, a full-screen countdown page appears. The user can click Continue anyway to skip the wait, or let the timer run out to proceed to the blocked-site redirect.

The Allow window setting lets the site through for N seconds after the countdown finishes — so repeated visits in quick succession don't keep triggering the countdown.

💡 Find Reset delay windows at the bottom of the Rules tab to clear all allow-windows.
Pomodoro timer — work sessions with automatic break unlocks
⏳ Focus & Timer tab → Pomodoro Timer

A built-in Pomodoro that cycles between Work sessions (rules fully active) and Break sessions (all rules suspended so you can browse freely). Set work and break durations in minutes, then click Start timer.

The cycle runs in the background — it continues even if you close the options page. The current phase and remaining time are shown in the Focus tab.

Lockdown mode — prevent disabling for a set duration
⏳ Focus & Timer tab → Lockdown Mode

Click Activate lockdown and type LOCK to confirm. For the chosen duration (up to 24 hours) you cannot turn off the extension, delete rules, or clear the list. A countdown shows remaining lockdown time.

To end it early you must first pass the PIN screen (if a PIN is set).

💡 Use this when you need to commit to a focus session and don't trust yourself to stay off distracting sites.
Disable delay — add a countdown before the extension turns off
⚙️ Settings tab → Disable Delay

Set a number of seconds (up to 300). When you hit the Disable button, blocking stays active for that long before actually turning off. An orange badge on the toolbar counts down. Click Cancel in the options banner to abort.

Custom blocked page — show your own message instead of redirecting
🚫 Blocked Page tab

Enable Use custom blocked page. Instead of redirecting to another site, the user sees a built-in "Site Blocked" page with your custom title, message, and optional image.

  • Title — shown at the top of the card (default: "Site Blocked")
  • Message — body text explaining why the site is blocked
  • Image — upload a PNG/JPG (max 1 MB) that appears above the card
  • Click Preview to see how it looks before saving
💡 Per-rule and per-group redirect URL overrides still take precedence over the custom blocked page.
Motivational quotes on the blocked page
🚫 Blocked Page tab → Motivational Quotes section

Tick Show motivational quote. A random quote appears at the bottom of the blocked page each time a site is blocked.

To use your own quotes, type them one per line in the Custom quotes textarea and save. Leave the textarea empty to use the built-in quote collection.

Stats dashboard — see which sites you tried to visit most
📊 Stats tab

The Stats tab shows blocking activity for the current week (resets every Monday):

  • Today — how many redirects happened today
  • This week — total redirects this week
  • Top blocked site — the rule that fired the most
  • Bar chart — daily redirect counts for the last 7 days
  • Top 5 list — your most-blocked sites this week

Hit counts per rule also appear as a small badge on each rule row. Click Reset hit counts at the bottom of the Rules tab to zero them all.

PIN lock — password-protect the settings page
🔒 Advanced tab → Security — PIN Lock

Set a PIN or passphrase. From then on, anyone who opens the Easy Redirect options page must enter it to view or change anything. The PIN is stored as a salted PBKDF2-SHA256 hash — the raw value is never saved.

  • Change password — expand the "Change password" section inside the active-lock panel
  • Remove lock — click Remove and confirm with your current password
  • After 10 wrong attempts the screen locks for 60 seconds
Add-rule friction code — force slow, deliberate rule changes
🔒 Advanced tab → Add-Rule Friction Code

When enabled, adding any new rule requires you to manually type a randomly generated code (32–256 characters, no paste allowed). This makes impulsive "just unblock it real quick" much harder.

Use the length slider to increase the difficulty. Longer codes take more time to type.

Import & Export — back up or share your rule list
📋 Rules tab → Import / Export (scroll down)

Export JSON saves all your rules, groups, settings, and theme as a JSON file. Export text saves one pattern per line for use with other tools.

Import JSON — merge mode adds new rules without overwriting existing ones (duplicates are skipped). Replace mode requires typing REPLACE to confirm, then swaps your entire rule list.

Import text — paste plain domains/patterns one per line.

Allowlist mode — block everything except your list
⚙️ Settings tab → Mode

Switch from Blocklist (default) to Allowlist. In allowlist mode every website is redirected except the ones on your list. Your list now means "these sites are allowed through". Everything else gets blocked.

Use this when you want to allow only a small set of sites (e.g. work tools) and block everything else.

💡 Switching modes never deletes your rules — they just change meaning.
Always-allowed list — sites that bypass blocking in both modes
⚙️ Settings tab → Always Allowed

Add patterns here that should never be redirected regardless of mode or rules. The extension's own options page is always in this list automatically.

Sub-resource blocking — block iframes too
⚙️ Settings tab → Block Sub-resources

By default, only top-level page navigations are blocked. Enable Block sub-resources to also block iframes (embedded pages) that load a blocked domain — useful for preventing embedded social feeds or video players from loading.

Redirect notification — get a toast when a site is blocked
⚙️ Settings tab → Redirect Notification

Tick Show notification on redirect to display a Chrome desktop notification each time a rule fires. The throttle setting prevents notification spam — e.g. set to 5 seconds to show at most one notification every 5 seconds.

Incognito support
⚙️ Settings tab → Incognito

Choose Block in incognito (default) or Allow in incognito. Block mode requires you to also enable "Allow in Incognito" for the extension at chrome://extensions — there is a direct link in this section to do so.

Profile name — label this Chrome profile's rule set
⚙️ Settings tab → Profile

Give this profile a display name (e.g. "Work" or "Personal"). The name appears in the page title and in the JSON export filename so backups from different Chrome profiles don't collide on disk.

Uninstall feedback URL
🔒 Advanced tab → Uninstall URL

Optionally set a URL that Chrome opens when the extension is uninstalled. Leave blank to use the default feedback form.

Keyboard shortcuts
🔒 Advanced tab → Keyboard Shortcuts · also: sidebar footer
  • Ctrl+Shift+B — open this options page from anywhere
  • Ctrl+Shift+Y — toggle the extension on/off
  • / — on the Rules tab, focus the search box
  • Esc — clear the search box

Click Customise shortcuts (sidebar footer) or go to chrome://extensions/shortcuts to change the key bindings.

Search & sort your rule list
📋 Rules tab → search box & sort controls

Type in the search box to instantly filter rules by pattern or group name. The sort dropdown lets you order by Newest, Oldest, A→Z, Z→A, or Most blocked. The ↑↓ button reverses direction.

Press / anywhere on the Rules tab to jump to the search box without clicking.

Context menu — block a site with right-click
Right-click any page → Easy Redirect

Right-click anywhere on a page to see the Easy Redirect context menu. Choose Block this site (blocks the whole domain) or Block this URL (blocks the exact path). A submenu lets you add it directly to a specific group.

A desktop notification confirms the rule was added.

Dark mode / Light mode / Auto theme
Sidebar → bottom-left toggle

Three options at the bottom of the sidebar: Auto follows your OS dark/light preference, Light forces the light theme, Dark forces the dark theme. Your choice is saved and applied immediately.
